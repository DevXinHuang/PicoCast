# Limitations

- This software emits research candidates, not operational aviation guidance.
- Bare picoballoon radar cross section is not assumed to be reliably detectable
  at all ranges.
- Biological targets, chaff, anomalous propagation, interference, sea spray,
  and residual clutter can mimic weak nonmeteorological targets.
- Scan cadence is measured in minutes, and beam volumes are large compared with
  a picoballoon.
- The first detector is interpretable and conservative. Supervised ML requires
  a labeled corpus that does not exist in this repository yet.
