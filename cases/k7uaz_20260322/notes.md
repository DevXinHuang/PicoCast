# K7UAZ 2026-03-22 Validation Case

This is the first PicoCAST validation case.

It uses K7UAZ launch-day pico-balloon telemetry from March 22, 2026. The track
starts near Tucson, Arizona and moves generally east during the available
launch-day telemetry window.

The main validation question is whether nearby NEXRAD Level II data contains a
compact non-weather candidate near the known balloon positions at the expected
times and altitudes.

This case should be used before attempting blind detection. It gives the
pipeline a known target track for checking time handling, geospatial alignment,
candidate scoring, and analyst review outputs.
