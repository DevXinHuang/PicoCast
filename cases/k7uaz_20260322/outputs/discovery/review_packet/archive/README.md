# Review Packet Archive

These files are preserved as intermediate review artifacts from the K7UAZ
candidate triage workflow.

Open `../index.html` first for the presentation-ready review website. The
archived files are useful when you want to inspect the earlier map-only outputs
or compare how the review packet evolved:

- `review_packet_gis_overlay.html`: full map-only GIS overlay for the review queue.
- `review_packet_gis_overlay_top_01.html`: focused map-only overlay for the top review item.
- `review_packet_tracklets.geojson`: GeoJSON export for review tracklets.
- `review_packet_tracklets_top_01.geojson`: GeoJSON export for the top review item.

Scientific status is unchanged: these are telemetry-consistent near-track radar
features requiring visual inspection and multi-radar confirmation.
